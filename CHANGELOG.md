# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-13

### Added

* Laravel 10 support.
* Laravel 11 support.
* Laravel 12 support.
* Blade renderer.
* Inertia renderer.
* API renderer.
* Log file parser.
* Multiple log file support.
* Log file selector.
* Log filtering by level.
* Log filtering by channel.
* Log filtering by date.
* Log filtering by exception presence.
* JSON context detection.
* Exception detection.
* Configurable middleware.
* Log clearing support.
* Configurable clear permissions.
* Support for Spatie Laravel Permission.
* Publishable configuration.
* Publishable Blade views.
* Publishable Vue/Inertia components.
* Automatic package discovery.

### Security

* Log file access validation.
* Log clearing whitelist.
* Middleware-based route protection.
